#BaijiahaoTerminater
>tampermonkey plugin  


Auto remove all content in search result of Baidu about Baijiahao,and other aside AD content in Baidu main site powered by Baidu CAS。
自动屏蔽所有百家号相关内容,百度合作推广网站搜索结果(如新闻类)，侧边栏推荐等
